# Kotlin_functions
Kotlin utility functions for solving certain problem sets.

# How to run the code ?
The easiest way is to copy the code into the Kotlin Playground website of jetbrains: https://play.kotlinlang.org/

# Other ways:
1. Either use InteliJ Idea IDE\
2. Download and setup the Kotlin compiler in your system by adding the compiler to the PATH environment variable, then use the terminal to run the Kotlin script.\
